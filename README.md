C++泛型编程，这是1980年C++诞生时创新的一大特性；使用C++(毋宁说，泛型编程)，可以在考虑算法时，不必考虑数据类型。所谓 “算法是对无数同类问题的一个有限可达解”。所以这个语法很重要。
